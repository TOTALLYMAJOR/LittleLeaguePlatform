# Season Archive Readiness Checklist

Use this before marking a season archived.

## Preserve

- Export roster, contacts, schedule, RSVP, snack, volunteer, sponsor, and notification CSVs through `/api/admin/exports`.
- Confirm exports are generated by an active organization admin and produce `admin_export_created` audit events.
- Keep teams, players, guardian links, events, RSVPs, media metadata, sponsors, notifications, and Parent Replay records readable after season close.

## Retain Or Delete

- Preserve non-chat season data for league records and family history.
- Run chat retention cleanup before archive proof.
- Confirm deleted chat message text is no longer present in app-readable `team_chat_messages`.
- Keep chat moderation audit metadata without reconstructing deleted message bodies.

## Prove

- Capture export filenames and row counts.
- Capture archive date, acting admin, and season id.
- Capture chat retention cleanup evidence and a read-only archived-season smoke check.
